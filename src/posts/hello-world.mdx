# Hello World: Why I Built This Site

After years of building things for other people, I figured it was time to build something for myself. This is that something.

## Who am I?

I'm Tufan — a software engineer based in London. I've spent the last few years working across the public and private sector, building everything from government workflow platforms to AI-powered chatbots. Currently I'm at JPMorganChase.

Before that I was at Transform, where I got to work on some genuinely interesting projects — leading front-end architecture for the Crown Prosecution Service, building data visualisations for the Department for Energy Security & Net Zero, and shipping campaign tools for Honda.

## Why a personal site?

Mostly because I wanted a place to write. I've been collecting thoughts on front-end architecture, testing strategies, and the realities of consulting work for a while now. Having somewhere to put them felt overdue.

I also wanted a space that wasn't LinkedIn or Twitter. Somewhere I actually own.

## The stack

I kept it simple:

- **Vite** for the build tooling — fast, no fuss
- **React + TypeScript** — what I reach for by default
- **Tailwind CSS v4** — the new CSS-first config is a nice step forward
- **shadcn/ui** — solid component primitives without the overhead of a full library
- **MDX** — blog posts are just Markdown files in the repo

No CMS, no database. I write a `.mdx` file, commit it, and it's live. That's the whole workflow.

Here's what a code block looks like, for the curious:

```tsx
function greet(name: string) {
  return `Hello, ${name}!`
}
```

## What I'll write about

I'm planning to cover things like:

- **Front-end architecture** — patterns that have worked (and ones that haven't) across real production systems
- **Testing** — practical approaches to testing React apps beyond the basics
- **Career stuff** — navigating consulting, switching jobs, and the SC clearance process
- **Side projects** — what I'm building and what I'm learning

No schedule, no promises. Just writing when something's worth sharing.
